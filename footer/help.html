Common Issues & Solutions

PDF Export Problems

If you're having trouble exporting PDFs:

  • Ensure your browser is updated to the latest version
  • Check if pop-ups are blocked for the site
  • Try using Chrome, Firefox, or Edge browsers

Calculation Errors

If calculations seem incorrect:

  • Verify all input fields are filled correctly
  • Check units and formatting of numbers
  • Refresh the page and try again

Data Not Saving

If your data disappears between sessions:

  • Data is stored locally in your browser
  • Clearing browser cache will delete your data
  • Export important data as PDF before clearing cache

Frequently Asked Questions

Are HSE Tools really free to use? +

Yes, all our basic HSE tools are completely free with no hidden costs. We believe in making safety tools accessible to everyone.

Do I need to create an account? +

No account required! All tools work immediately without any registration. Your data stays locally in your browser.

Can I use these tools for commercial purposes? +

Yes, you can use our tools for both personal and commercial projects. The generated reports are professional and audit-ready.

How often are new tools added? +

We regularly update and add new tools based on user feedback and industry requirements. Check back frequently for new features!

What browsers are supported? +

HSE Tools work on all modern browsers including Chrome, Firefox, Safari, and Edge. For best experience, use the latest browser versions.
